2) timedatectl set-ntp true

3)Verificar la tabla de particiones:

4)gdisk /dev/sda
Se debe listar “GPT Present” al final de la lista.

5)Crear particion swap :

gdisk /dev/sda

7)Crear partición /home :

gdisk /dev/sda

Verificar:

lsblk

8)Formatear particion swap :

9)mkswap /dev/sda5

10)Activar swap :

swapon /dev/sda5


11)Formatear particion / :

mkfs.ext4 /dev/sda6


12)Formatear partición /home :

mkfs.ext4 /dev/sda7


13)Montar particion / en /mnt :

mount /dev/sda6 /mnt

14)Crear directorio para /boot :

mkdir -p /mnt/boot

15)Montar partición /boot, en este caso es /dev/sda2 pero puede cambiar, se debe usar la partición EFI de Windows:

mount /dev/sda2 /mnt/boot


16)Crear directorio para /home :

mkdir -p /mnt/home

17)Montar partición /home :

mount /dev/sda7 /mnt/home


18)Instalar los paquetes base:

pacstrap /mnt base base-devel linux linux-firmware nano

Esto iniciará la instalación de los paquetes base (191.35 MiB aprox.)


19)Generar fstab:

genfstab -U /mnt >> /mnt/etc/fstab


20)Verificar:

cat /mnt/etc/fstab


21)Iniciar sesión como root en la instalación:

arch-chroot /mnt /bin/bash


22)Generar locales:

nano /etc/locale.gen


23)Descomentar las líneas de interés quitando el símbolo #, en este caso:

en_US.UTF-8 UTF-8
Guardar presionando Ctrl + X, luego Y y finalmente ENTER

24)Construir el soporte de idioma:

locale-gen

25)Crear el archivo de configuración correspondiente:

nano /etc/locale.conf

26)Agregar el siguiente contenido:

LANG=en_US.UTF-8

Guardar presionando Ctrl + X, luego Y y finalmente ENTER

27)Ajustar zona horaria:

tzselect
2 
ENTER
14 (Número correspondiente a la zona)
ENTER
1 (Número correspondiente a la subzona)
ENTER


28) Borrar el archivo de configuración anterior y crear el link simbólico para hacer el cambio permanente:


rm /etc/localtime
ln -s /usr/share/zoneinfo/<ZONA>/<SUB_ZONA> /etc/localtime
donde < ZONA > puede ser America y < SUB_ZONA > puede ser Bogota.

29) Instalar systemd-boot:

bootctl --path=/boot install


30) Generar archivo de configuración de systemd-boot:

nano /boot/loader/loader.conf

31) Agregar el siguiente contenido:

default arch
timeout 3
editor 0
Guardar presionando Ctrl + X, luego Y y finalmente ENTER

32)Generar el archivo de la entrada por defecto para systemd-boot:

echo $(blkid -s PARTUUID -o value /dev/sda6) > /boot/loader/entries/arch.conf
Esto generará un archivo de nombre arch.conf en la ruta especificada, con un contenido similar a:

14420948-2cea-4de7-b042-40f67c618660


33)Abrir el archivo generado:

nano /boot/loader/entries/arch.conf


34 )Se debe agregar lo siguiente, de manera que el serial generado, quede después de PARTUUID y antes de rw, como sigue:

title ArchLinux
linux /vmlinuz-linux
initrd /initramfs-linux.img
options root=PARTUUID=14420948-2cea-4de7-b042-40f67c618660 rw

guardar

systemctl enable NetworkManager
